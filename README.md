# 项目简介

## 核心功能
- 管理员管理：系统管理员账号、权限、信息管控维护
- 客户管理：客户信息录入、查询、修改，客户情况等
- 问卷管理：问卷创建、编辑、发布，问题、选项定制
- 标签管理：业务对象标签分类
- 公告管理：系统公告发布、编辑、删除
- 学习资源管理：上传、整理、分类学习资料
- 友情链接管理：添加、审核、删除外部友情链接
- 留言管理：用户留言集中处理，便于查看、回复

## 适用场景
- 教育机构：用于调查分析学生学习资源需求
- 企业培训：评估员工学习资源利用情况
- 研究机构：收集相关学习资源使用反馈

## 技术特点
- 采用Springboot框架开发，确保系统稳定性与易用性
- 基于MYSQL5.0数据库，数据安全可靠
- 支持Java开发环境，兼容性高
- 配套万字文档，方便开发者快速上手
